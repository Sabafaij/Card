Technologies Used

Introduction to HTML

HTML is the standard markup language used to create and structure web pages using headings, paragraphs, links, images, and forms.

Introduction to CSS

CSS is used to style and design web pages by controlling layouts, colors, fonts, spacing, and responsiveness.

Introduction to JavaScript

JavaScript adds interactivity to websites by enabling animations, validations, dynamic updates, and user interactions.

Introduction to Bootstrap

Bootstrap is a powerful CSS framework that helps build responsive, mobile-first websites using ready-made components.
